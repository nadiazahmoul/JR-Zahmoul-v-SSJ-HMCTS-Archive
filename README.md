# JR-Zahmoul-v-SSJ-HMCTS-Archive

This repository is a public transparency archive relating to the Judicial Review
**Zahmoul v Secretary of State for Justice & HMCTS**
(Administrative Court reference: AC-2025-LON-004293).

The archive preserves a timestamped record of:
- court filings and submissions,
- court orders and judgments,
- correspondence with HMCTS, the Ministry of Justice, and public authorities,
- pre-action correspondence and escalation stages,
- evidence bundles and exhibits,
- medical participation material,
- and public-interest / oversight communications.

Each numbered folder corresponds to a distinct procedural or evidential category.
See individual folder README files for scope and document descriptions.

This repository is maintained for transparency, accountability, and ease of reference
by the Court, oversight bodies, journalists, and researchers.
