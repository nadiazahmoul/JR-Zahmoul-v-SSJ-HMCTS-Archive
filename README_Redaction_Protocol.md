Public Document Redaction Protocol

JR-Zahmoul-v-SSJ-HMCTS GitHub Archive

1. Purpose and governing standard

This protocol governs the redaction of documents uploaded to the public GitHub repository associated with Zahmoul v Secretary of State for Justice & HMCTS (Administrative Court ref. AC-2025-LON-004293).

Governing standard:

Documents are redacted only to the extent necessary to comply with
(a) data-protection obligations, and
(b) established court publication practice,
while preserving evidential integrity, accountability, and public-interest transparency.

Redactions are minimal, rule-based, content-neutral, and reversible.

2. Scope

This protocol applies to:

Exhibits

Evidence bundles

Correspondence

Schedules

Medical or safeguarding material

This protocol does not apply to:

Court orders or judgments already published unredacted by the court

Legal argument where no personal data is present

7. Versioning and reversibility

For every redacted document:

An unredacted master copy is retained offline

A clearly labelled public version is published

File-naming convention
YYYY-MM-DD_[Document_Description]_Redacted_Public_Version.pdf


Example:

2025-12-01_Exhibits_Redacted_Public_Version.pdf


This ensures auditability and reversibility if ordered by the court.

8. Repository disclosure statement (README language)

Folders containing redacted documents should include (or reference) the following statement:

Documents in this folder are redacted public versions prepared in accordance with a rule-based redaction protocol. Redactions are limited to personal data and non-public identifiers, consistent with court publication standards. No evidential, procedural, or accountability-relevant material has been removed.

9. Presumption rule (critical)

When in doubt:

Redact identity, not substance.
Preserve facts, actions, dates, responsibility, and chronology.

10. Rationale

This protocol:

aligns with court publication practice

complies with GDPR minimisation

avoids selective or narrative redaction

preserves evidential weight

protects third parties

supports public-interest transparency

It can be justified in one paragraph if ever challenged.
